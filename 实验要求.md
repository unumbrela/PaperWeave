Web应用开发技术

 

**一、实验目的**

1、综合运用在本课程内外学到的Web应用开发技术，独立设计并实现一个具有实际应用价值的Web应用；

2、掌握Web前后端开发的基础知识，包括HTML、CSS、JavaScript、后端编程语言、数据库技术等；

3、通过大作业的实践，加深对Web应用开发流程的理解，提高解决实际工程问题的能力。



**二、实验要求**

1、设计和实现一个任意类型的Web应用，满足基本的功能要求；

2、使用的前端技术不限，前端界面要求界面美观、简洁、易用，布局合理，适配主流浏览器和不同分辨率的设备，可使用前端框架来提高开发效率和代码的可维护性；

3、选择任意一种后端编程语言来开发 Web 应用的后端服务，可使用 Web 框架来简化后端开发流程，同时使用数据库技术来存储 Web 应用的数据；
